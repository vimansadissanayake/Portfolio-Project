Personal Portfolio Website

A clean, modern, and responsive personal portfolio website showcasing my work as a UI/UX Designer and Front-End Developer.

Overview

This is a single-page portfolio built with pure HTML, CSS, and a touch of JavaScript. It features:

- Hero section with animated typing effect
- About me section
- Skills (Frontend, UI/UX, and Certifications)
- Featured projects with GitHub links
- Contact information
- Smooth animations and hover effects

Dark-themed design with blue accents.

Technologies Used

- HTML
- CSS
- JavaScript


How to Run

Simply open `index.html` in any web browser.
No build tools or dependencies required.
